Tipos de datos en JavaScript

JavaScript tiene varios tipos de datos, incluyendo:

  • Números: 123, 3.14
  • Cadenas de texto: "Hola", 'Mundo'
  • Booleanos: true, false
  • Objetos: { nombre: "Juan", edad: 30 }
  • Arreglos: [1, 2, 3, 4, 5]
  • Arreglos de cadenas: ["Moone","Ximena","Emma","Sofia","Valentina"]
